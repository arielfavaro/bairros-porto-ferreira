Metodologia de Geração dos Resultados

Os dados utilizados foram extraídos do Cadastro Nacional de Endereços para Fins Estatísticos (CNEFE) do IBGE, ano base 2022.

Inicialmente, os dados foram importados para um banco de dados PostgreSQL, com suporte à extensão espacial PostGIS. A partir disso, foi realizada uma análise de clusterização espacial utilizando o algoritmo DBSCAN, considerando agrupamentos de pontos com distância máxima de 500 metros entre si e contendo no mínimo 15 registros por agrupamento.

Para cada cluster identificado, foi gerada uma geometria aproximada utilizando o método Convex Hull, representando a área de concentração de endereços por localidade.
